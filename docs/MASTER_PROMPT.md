**My Requirements:**

3. **The Engagement Loop** — Define a 3-step Core Loop that ensures users return daily (e.g., Daily Quests, Streak Rewards, and Faction Leaderboards).

4. **Technical Stack** — Recommend a tech stack suitable for a real-time, location-based app that supports high-frequency updates, focusing on scalability and low latency for the Kenyan mobile market (consider Python, Flutter, or relevant location-based APIs).

5. **Monetization (Ethical)** — Suggest a monetization model that feels native to the game (e.g., "Campus Currency" for profile customization or access to exclusive "Networking Zones") without alienating student users.

Please present this in a structured, professional format, keeping in mind the target demographic is university students in Kenya.
